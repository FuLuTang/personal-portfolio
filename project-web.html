Tourism Website 旅游网站前端 Intégration Web Front-end

HTML5 CSS3 Responsive

Creation of a website in a group, using HTML and CSS. This is a web development project to present a tourist site. Realized in collaboration with two classmates, I was responsible for the "Contact" page, the global CSS style, as well as the navigation bar and the footer. The site is responsive, adapting dynamically to different window sizes, and has passed W3C certification.
