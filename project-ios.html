iOS Task App iOS 任务管理应用 Application Mobile iOS

SwiftUI iOS Productivity

This is a task management software. It allows tasks to be executed more efficiently by classifying them into 4 categories according to their urgency and importance. This software was developed entirely by myself using Apple's SwiftUI. This project reinforced my skills in User Interface (UI) design, front-end development, and data structure design. A publication on the AppStore is planned. 这是一款任务管理软件。它根据紧急程度和重要性将任务分为 4 类，从而更高效地执行任务。该软件完全由我使用 Apple 的 SwiftUI 开发。该项目增强了我在用户界面 (UI) 设计、前端开发和数据结构设计方面的技能。计划在 AppStore 上发布。 Il s'agit d'un logiciel de gestion des tâches.
