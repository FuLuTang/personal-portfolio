2D RPG Game 2D RPG 游戏 Jeu Vidéo RPG 2D

C++ SDL2 Game Engine

I developed a 2D RPG game engine autonomously using C++ and SDL2. This project involved implementing a rendering engine from scratch, designing complex enemy AI logic, and creating a dynamic camera system that follows the player smoothly. 我使用 C++ 和 SDL2 自主开发了一个 2D RPG 游戏引擎。该项目涉及从零开始实现渲染引擎，设计复杂的敌人 AI 逻辑，以及创建一个能够平滑跟随玩家的动态摄像机系统。 J'ai développé de manière autonome un jeu RPG 2D en utilisant C++ et SDL2. Ce projet comprenait l'implémentation d'un moteur de rendu, la conception d'une logique d'ennemis complexe et la création d'un système de caméra dynamique qui suit le joueur de manière fluide.

Key Features 主要功能 Fonctionnalités Clés

  • Custom Rendering Engine with SDL2. 基于 SDL2 的自定义渲染引擎。 Moteur de rendu personnalisé avec SDL2.
  • Enemy AI with state machines. 基于状态机的敌人 AI。 IA ennemie avec machines à états.
  • Dynamic Camera System. 动态摄像机系统。 Système de caméra dynamique.
